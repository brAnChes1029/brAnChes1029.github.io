終於要踏上國手之路了！！！(☆▽☆)

02/02 (Mon.)

因為 8.30 要報到，所以 6.20 就要起床，不然我爸說會塞車。
然後我妹也跑過來，到底想幹嘛。
結果 8. 就到了，會不會太早。

這次 EGOI 和之前 TOI 還是鬥 code 都不一樣，是在另一個教室，在之前那個館的正後方，TOI 辦公室對面。
然後我又坐第一排了，算運氣很好嗎？
接下來就大概有一些人陸續過來了，北一女超級多欸，原因不明；還有一些感覺是 EGMO 過來的，有個 EGMO 雙金牌耶，我也想要牌啦。
房間的部分我是跟王禹喬和另一個北一女的。

然後就是 Greg Lee 講話，他說 EGOI 終於辦成了，但沒有經費所以去找了中技社的金主贊助我們，還說選上國手一定要出國 ( 拜託我超想去的好嗎 )。
然後我從學姊那裡聽說 TOI 被砍經費所以住師大會館，之後 TOI 可能也要住師大會館。( 我跟江語萱說的時候她回了一個骷髏頭 )。

9.41 開始我和楊岳蓁講初選，她講到 10.01 所以我還要講 $29$ 分鐘，雖然我一到就在到處喊很難但還是順利講完了，大家都不敢舉手說有問題，很棒。
講完大概 10.30 是初選補題，我很閒把 TOI 之前潛力組的 123 木頭人 寫掉了，我覺得很好玩欸，好題大推。
然後我又再去寫了一次全國賽寫不出來的 SPFA，然後發現負環細節也太多。
我還寫了 SCC，因為我好像快忘記了。

接下來就是吃午餐，是師大學餐。
大家都一群一群好煩喔，交朋友好難喔，江語萱來陪我 QAQ

下午大概 13.30 開始練習，因為都是鬥 code 題然後寫我會的感覺意義不大，所以就來寫 CSES，但我覺得我還是在浪費時間。
接下來就是侯欣緯回來經驗分享，好像和 TOI 2025 差不多，不過我有用 slido 問一些問題。
比較重要的大概就是說要穩定發揮 ( 基本題都要好好做出來 )，然後可以想難題是看到什麼讓我想到怎麼解，至於 vir OI 的部分和我之前認知差不多。

約 17. 吃飯，是吃跟鬥 code、TOI 一樣的外燴。
吃完大概 18. 開始和媽媽講電話，然後差不多 18.50 就準備收東西回飯店了，然後我們是有車可以搭回去的喔。

回到飯店大概 19.20，拜託這真的是飯店嗎，有點小行李箱放不下和有點髒先不說，最可怕的是那個浴室，浴室到底在哪裡啦，根本只有一個蓮蓬頭，連個擋水的簾子都不給，根本 💩，而且那個浴室超級小，一洗澡就會整間超級濕，重點是地上有很多謎之屑屑，超級恐怖 💩💩💩
我要給他 $3.8/10$ ，一樓有抹茶販賣機所以幫他加 $1$ 分變 $4.8/10$

洗完澡大概 21. 我就開始寫圓錐曲線，然後對每個題目亂爆開，大概 22.30 開始準備睡覺，但好像 23. 多才關燈，然後隔壁 509 到底是誰一直在吵，還有一個機器會發出謎之低頻噪音，而且床超級小，超級難睡。

忘記說網路了，網路也超爛，洪瑋澤沒有騙我欸，所以我果斷打開自己的網路。

要不是為了當國手誰要住這種破地方 ￣へ￣

02/03 (Tue.)

7.50 要集合所以我大概 7.30 起床，但我中間可能 3.10 、 6. 之類的已經醒來過好多次了，超級累。
8.20 到教室開始吃早餐，我是點吐司肉排蛋加紅茶，但他的吐司質感很怪，不是鬆軟那種，就是有點難咬然後有點韌性，不會形容。

上午教 Greedy、BFS、DFS、MST、Dijkstra 什麼的，所以我開始打 JOI 2025 予選 2 ( 因為我不知道那是啥，感覺很好玩 )。
但因為精神狀態有點糟，一直不會減法，還有不會對單峰函數斜率變化二分搜，慘。

午餐照吃師大學餐，下午繼續把沒打完的打完。
我覺得 p6 卡不掉 $O(N^3)$ 有點扯，但有可能是因為一些數學性質所以構不出來 ( ? )

喔對下午睿睿說因為 IOI 沒有 codeblocks 所以之後模考都不會有，我去說 EGOI 2025 是有 codeblocks 但還是難以挽回，所以要練習用 vscode ，好煩喔。

接下來我就在裝 vscode，然後一直爛掉，我就去找學姊來幫我裝，但是 debugger 還是一直爛，到現在都沒有修好。
所以我就心情很差，跑去資工系館外面散步，結果遇到一個大笨鳥，是說大笨鳥真的很會挑時間出現欸，他都會挑你心情不好的時候，然後一直在那邊搖脖子，超級可愛，像是在跟我說 EGOI 國手穩啦不用擔心。
學姊也說我模考要是不會用還是有操作上的障礙，她都會來救我，看來是可以相信她，然後不要擔心太多。

回來之後晚餐有超多櫛瓜，超級棒。

跟媽媽打完電話大概 18.30 要出發去體育課，當然我也是一路在旁邊水到底，然後就回師大會館了。

回去大概 20.40 我出發去找耳塞，但便利商店沒有，所以改去師大夜市吃雞蛋糕 ( 因為學姊說好吃 )，然後我還成功使用了 line pay ，但只能說我比較喜歡吃外面酥酥的那種，沒有很喜歡軟軟的厚實型那種。
我想念紫米牛奶了啦。

意外的是今晚沒有特別吵，然後有穿拖鞋的我變身超人成功抵擋所有屑屑的攻擊，還有我覺得睡前聽點歌是對睡眠有幫助的。

02/04 (Wed.)

一樣 7.50 起床然後坐車，我覺得在車上睡覺挺不錯的，然後今天早餐是玉米蛋餅。

吃完可能 9. 我開始打 JOIGSC 2023 day1，第一題因為二分搜時把 ans=mid 打成 ans=l，WA 到懷疑人生，超級大失誤。
第二題就是拓樸 BFS，啊但是有點小頭暈。
第三題我只會做 $58$，而且我到 19. 都還看不懂他的解。
第四題可能只有 $25$。
所以這樣是 100/100/58/25，可能有日本前 $3$ 嗎，敗筆就是沒好好想第四題。

是說上午的教授有個很怪的地方是他要我們寫一個 tree DP，然後把 code 傳給他，但是沒給 judge，所以他要用眼睛看是不是對的。

午餐吃完我去師大圖書館看漫畫，我覺得圖書館很棒欸，好多漫畫喔。
13.25 江語萱說她 TMO 考完了，所以我就跑去理學院門口找她，結果她在大門口，所以我就又跑過去，然後她給我一個抹茶牛嘎糖 ( 我不會打啦 ) 我就又跑回去教室了，到教室的時候已經 13.37 了。

下午題目是 DP 但好像還好，樹最小獨立集我小時候可能疑似寫過，但還是花一點時間才想起來怎麼做，然後這題好像沒有人會做。
因為我還是不會上午第三題，加上我很無聊所以又去煩輔導員，但她當然也不會所以叫我去找 wiwiho，但我覺得也太高攀不起了，所以她說要幫我問，但她忘了。

我記得下午遇到睿睿問我師大會館怎樣，我說要是沒當上國手我會後悔我住過師大會館，就是我已經抱著破釜沉舟的決心來住師大會館了，一定要給我當國手。

吃完飯我就打給媽媽說有點無聊，因為我一直不會第三題，所以我決定打完這個 blog 就要去圖書館放鬆一下。

大概 18.40 我就到了圖書館，因為我學姊之前跟我說二樓有鋼琴，所以我決定去一樓登記然後借耳機，但是非師大學生不能彈，喔不。
所以我就去看漫畫了，這裡有芙莉蓮欸，好好看。
看著看著我就發現我媽一直打給我，而且超級多通，沒想到是因為我忘記講去哪裡了，害睿睿被我嚇死以為我不見了，好啦我下次會記得講，造成大家的恐慌我很抱歉。

回去教室之後就 19.40 了，然後就差不多可以搭車回去了。

然後今天晚上我做了一個噩夢，就是我基於不明原因沒有考倒一模和二模，所以輸國手線超多分然後要靠三模和四模拉回來，聽起來至少有 $1e5$ 個不合理的地方，但反正我是被嚇到了。

02/05 (Thu.)

今天早餐是漢堡，據說是雞排蛋漢堡，但我不知道雞排去哪了，為什麼變成一個像是壓扁的雞塊那樣的怪東西。
喔還有昨天下午有人換位子坐到我旁邊，我旁邊原本是空的，但她看不到，喔然後她換過來之後還是看不到。

大概 9. 開始打 JOIGSC 2023 day2，看過一輪感覺 70/100/25/19，所以先做第二題。
第二題就是 DP，先傳一次驗解之後用 deque 優化掉一維就成功拿到 $100$。
第一題我覺得我的 Greedy 是對的，弄一下也 $100$ 了。
第三題我真的只會暴力，第四題我想一下覺得就至少 $N<=3$ 的 $19$ 分可以拿，之後就不確定了。
所以這樣是 100/100/25/19，可能有日本前 $2$ 嗎。

吃完午餐看我推的孩子，之後我要當我推的國手 ( ? )

下午開始看解，是說第四題也太有料了吧，超好玩構造法，但 $N>39$ 好像需要隨機，用一個叫模擬退火的演算法，酷。

接下來睿睿說不知道下午程式演練會不會算分，目前可能不會但未來要再討論，代表說萬一要計分但我都沒寫欸，燒雞。
第一階段平時成績是 $40 \% \times 20 \% = 16 \%$，所以程式演練可能最多有 $8 \%$，因為還有早上的成績，但是 $8 \%$ 有點多，燒雞。
所以我決定要來寫，當然不是我寫啦，我都用我小時候寫的 code 或是 AI 或是網路別人的 code，身為一個國手我很抱歉，但是我不想寫 UVA 怪題。

吃晚餐前我有去拼拼圖，我都用眼力大法找某塊拼圖是在哪裡，我覺得很棒。
晚餐吃咖哩欸，讚。

約 18. 吃完晚餐我打給媽媽，但她一直在幫我妹慶生然後都不跟我聊天，所以我就去看漫畫了 ( 我有和睿睿說 )。

看完漫畫大家在上體育課，我就把拼圖拼完了。
因為師大操場好像不給我們開燈，所以體育課在室內，也就是拼拼圖和吃飯的房間。
然後拼完拼圖我又打了一次電話。

20.40 回師大會館之後我去買可麗餅，爸爸告訴我的，這家店有點隱蔽欸，找好久，大概 $5$ 分鐘。
可麗餅滿大的也很好吃，我是吃巧克力 $55$ 元，然後在等餐的時候會招待冰淇淋喔。

上午把昨天第四題先寫到 $88$ 分，我覺得很好玩！！然後我會用 vscode 寫互動題了，但並沒有比較好用。
希望一模不要 我 vs vscode。

是說我都沒和侯欣緯講到話，後悔，但我感覺她和我的差距有 $1e18$ 欸。

吃完飯我決定要專心耍廢，先是邊吃邊玩電路遊戲，又找楊岳蓁和常貽媗和輔導員玩拉密 ( 輔導員贏了 )，再去揪人玩狼人殺，結果揪到 $12$ 個人，可以玩 $12$ 人局，超好玩，雖然我都靠場外殺人嘻嘻。下次應該早點揪人玩狼人殺的，後悔。

結果玩的時候超開心，回去就開始緊張，但經過和媽媽講電話以及調適後就還好了。
