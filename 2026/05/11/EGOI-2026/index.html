EGOI 2026
May 11, 2026

5/11 (Mon.)

登機前

我們是台灣晚上 23:15 的飛機，睿睿要大家 20:45 機場集合之類的，反正大家都很準時然後拍個大合照就出發了。
因為第一次搭這麼久 ( $14$ 小時 ) 的飛機所以我有點緊張，然後我爸媽和我妹都有來送機。
這次同行的除了四位國手，還有李忠謀和他老婆，和兩位教授。
過完海關 ( 海關叫楊岳蓁底迪 ) 之後李忠謀講了一點久義大利的扒手，之後就放我們去逛大街了。

飛機

上飛機之後我就找了一百公尺來看，但是大概台灣 1:00 多餐就出現了，吃一吃就關燈了 ( ? )，陳瑀萱說就跟著飛機的燈睡覺時差自然就調好了，不用憋到義大利時間晚上十點再睡，所以我就開始睡覺了。
我睡了台灣時間 2:00~3:50、4:30~7:50、8:50~11:15，但唯一有睡著的只有中間那段，因為實在太難睡了，我爸給我的頭枕超難用，睡一睡就會腰酸背痛脖子酸和腳麻，就慶幸我有早點睡覺和有存歌在手機裡面。

下飛機和接駁車

下飛機 ( 義大利 7:40 ) 過義大利海關，應該還滿快的，還有很多酷酷的自動化設備。
之後坐一小時的車到飯店，接駁車跟台灣遊覽車很像，不得不說他舒適度優超飛機，非常好睡。
另外得知有料消息鄧涵云爸媽陪她來義大利，超酷；以及李忠謀有美國護照，原來他跟常貽媗是一掛的。

飯店

今晚住 Crowne Plaza Milan，$2$ 人房 ( 我跟陳瑀萱，因為她說她可以陪我早點睡 )，這個飯店頂樓有健身房和桑拿和免費的水，房間裡面也有水啦，但只有兩瓶，然後冰箱不能碰，因為碰了要收錢。
另一個很酷的是他的廁所。

米蘭旅遊

大概 11:00 準備搭捷運去米蘭大教堂，順便吃午餐。
大教堂前面有超多鴿子會低空飛行差點撞到人有點恐怖，但更恐怖的是塞給你鴿子飼料再勒索錢的人，有個人試圖塞給李忠謀鴿子飼料但是失敗了，哈哈哈。
午餐吃義大利麵包，口味超多的根本不知道要點什麼，我看到某個有櫛瓜就直接點了，雖然我討厭吃麵包，但這個還滿好吃的。
吃完李忠謀就放生我們去亂逛，我們去買了紀念品和吃冰淇淋然後走一走之後就累了，所以就提早回去飯店休息了。
然後哪裡的捷運都差不多欸，根本不可能迷路啦。

休息和晚餐

我們大概 15:00 就回到飯店休息了，陳瑀萱說坐太久飛機了所以做了一個小時的瑜珈，太強了。
下午我們還去逛義大利的家樂福，買東西給同學之類的，她們都買了一堆開心果巧克力餅乾和千層酥，我覺得之後還有機會買啦，所以只買 $0.99$ 元的義大利麵一盒。
18:15 跟李忠謀去吃披薩，他可能覺得我們是大胃王然後就點超級無敵多，果真我們就不負眾望的吃不完了。
我覺得義大利燉飯比台灣的硬欸，還有陳瑀萱點的開心果麵疙瘩 ( ? ) 和海鮮義大利麵 ( 用麵皮包起來的 ) 特別酷。

5/12 (Thu.)

早上

昨天晚上洗完澡我就太想睡覺了，就不小心穿著浴袍睡著了 ( ? )，幸好陳瑀萱洗完澡有叫我起來吹頭髮刷牙，才避免憾事 ( 感冒 ? ) 發生。
所以我順利從 22:30 睡到 8:30，精神非常好，看來我的時差已經完全好了 (❁´◡`❁)
這個飯店的早餐超好吃的，然後我們待會要搭車去 EGOI 大會，沒有午餐吃。
李忠謀從今天開始要跟我們分道揚鑣去開會 ( ? )，好像是閉幕那天才會再看到他。

火車

我們坐 12:35 的車從米蘭去 Rimini，之後會有 EGOI 的車接我們去大會。
因為太早到車站了我們就一群人圍一圈玩 faamily style，但我的網路特別烏龜，別人的菜丟給我就不見了。
義大利的火車 ( 還是高鐵 ? ) 也很酷，位置有四個人面對面的，還有一個桌子在中間，而且窗戶前面還有垃圾桶。
我在打 blog 和打 code 的時候發現其他人都在睡覺了。
門旁邊那個義大利人的頭髮真的好酷喔，他的頭髮是天生就那麼捲嗎。

Rimini Station

15:00 我們到了 Rimini Station，一出站就看到兩個 receptionist 迎面而來，穿很橘的 EGOI 衣服，拿著 EGOI 的牌子。
Receptionist 其中一個是 Luca，另一個是他的朋友 ( 但我不知道他叫什麼，他沒戴名牌 )。
因為有人 delay 之類的我們要再等一陣子接駁車，所以就跟他們聊天。
他們兩個都在比薩念大學然後會一點中文 ( 我覺得他們講得很不錯 )，其中 Luca 的朋友很會跟我們聊天，Luca 是 IMO alumni ( silver & gold ) 然後不太會笑，我還被 Luca 的朋友認證有義大利文天賦 ( 但我就只是裝他們的口音而以欸 )。
我們聊到一半就突然出現一個瘋人一直對我們大吼大叫，但 Luca 他們超強把我們檔在後面，結果我們之後去跟教授講話時那人又出現了，我覺得他離我太近了很可怕所以就用很快的速度跑走了，好恐怖喔。
之後車就來了，我們就跟一個盧森堡領隊一起去 Cesenatico 了 ( 大概半小時就到了 )。

EGOI 飯店

EGOI 參賽者飯店是 San Pietro，教授們跟我們住不同地方，然後聽說他們的飯店沒有吹風機。
抵達飯店之後 check in 會拿到 EGOI 大會送的包包，裡面有 EGOI 帽子 ( 但超小的欸 )、貼紙、衣服 ( 淺藍色的，正面有小 EGOI，背面有很多贊助商 )，和贊助商送的桌遊、筆記本之類的紀念品。
我們還有跟香港隊交換禮物，拿到資料夾和 AC、WA、TLE 的貼紙 ( 顯然 AC 要貼在我這，其他拿去貼別人電腦 )。
跟澳洲隊講話的時候她們會偷偷把一個無尾熊夾到你的包包上，我是之後才發現的，然後只有我跟陳瑀萱有中獎，超酷。
我在大廳看手機的時候還莫明其秒撿到一個瑞士人丟過來的巧克力，滿好吃的。

因為比較早到，我們還跟美國人 ( 對，就是常貽媗她們 ) 一起去海邊，我們的飯店一出去就是海邊，很酷喔，然後我們就拍拍照和在沙灘上寫 EGOI。
晚餐是 19:30 開始在飯店的餐廳吃，餐都很酷，像是很像披薩的麵包、一塊超大的魚加番茄醬、各式義大利麵、我喜歡的櫛瓜、花椰菜馬鈴薯塔，餐廳還有免費的水，不然要買水的話要 $1.5$ 塊好貴。

參賽者的飯店比較小，感覺比師大會館大一點點而已，但浴室是乾濕分離、陽台風景很好 ( 但很冷 )。
床是兩張雙人和一張上下舖，我睡中間的床，楊岳蓁睡靠窗的床，陳瑀萱睡下鋪，鄧涵云睡上面。
插頭好像只有 $3$ 個而且其中一個很鬆，充不到電。
來歐洲之後我發現大概 19:00 之後就會覺得特別累，但我時差應該是調好的啊，所以為了不要讓時差功虧一簣我都會到 22:00 多再睡覺。

5/13 (Wed.)

早餐

9:45 集合出發所以我 8:00 起床，因為大家 ( 除了楊岳蓁 ) 都比我早起，所以很快整理一下就可以去吃早餐了。
早餐是在飯店一樓另一個餐廳吃的，大部分都是麵包和甜的派，甚至沒有歐姆蛋，我覺得有點不習慣但還好啦。
這個餐廳有點小，位子也很少，我們還差點沒位子坐。

practice session

因為大家都不太習慣所以很快就吃完早餐回房間了。
practice session 的地方跟明天比賽地方一樣在 colonial AGIP，飯店出去走路 $3$ 分鐘就會到了。
比賽會場在海的旁邊，有一面是窗戶，可以看到外面沙灘 ( 但感覺電腦反光很嚴重 )，反正我是坐在比較裡面的地方。
比賽的電腦鍵盤比李忠謀給的好用，所以我可以直接用他筆電鍵盤，然後位子不算小，還不錯。
他有給四個牌子 : 電腦壞掉舉紅牌、廁所舉藍牌、計算紙舉白牌、要食物要水舉黃牌，我在測試的時候不小心把藍牌弄斷了，我很抱歉。
廁所在會場外不會很遠的地方，滿乾淨的，但好像只有 $5$ 間 ( 有兩邊不同的廁所，所以總共有 $10$ 間 )，希望明天不要卡太久。
我有測試一般題和互動題，vs code 也好好的，還順便想出之前 virtual 的時候沒做出來的某個子題 ( 但滿分我還是不會 )。

把餅乾和奇異鳥娃娃留在那邊之後我就回去吃飯了。

下午

practice 結束大概 13:00 吃午餐，午餐跟昨晚晚餐差不多，我覺得貝殼義大利麵超好吃的，然後有烤櫛瓜耶，好好吃。
開幕式原定 14:00 但他一直 delay，所以我們還跟教授們去了附近一個超市，陳瑀萱問店員某兩種餅乾差別在哪，店員推薦巧克力比較多的那種，所以大家都買了一堆，我還買了另一種上面有很多星星的巧克力餅乾打算回去請同學吃。
我們還去了一個小紀念品店，買了一個有 Cesenatico 風景的磁鐵 ( 所以哪裡有賣明信片，我想要明信片！)
