EGOI 2026 後日談
Jun 12, 2026

這裡是 EGOI 2026 後日談。
大概是一點心得。

我大概從 EGOI 回來不久就有多少打一點，不過我最後決定押 6/12 這個日期。

注意注意

帶東西

  • 護唇膏之類的保濕東西 : 因為歐洲比較乾，至少義大利滿乾的，嘴唇裂開有點痛
  • 長袖長褲外套 : 歐洲緯度比較高，加上 2026 在海邊又去山上玩有點冷，不過還是要考慮時間地點啦，但保暖帶了不虧
  • 餅乾 : 除了賽中可以吃，如果吃不慣當地食物或搭飛機肚子餓都可以吃 ( 不過賽中可以學學陳瑀萱多拿一點大會準備的當地餅乾回家吃 )
  • 交換禮物 : 可以多花點心思準備要跟外國隊交換的禮物，台灣硬幣可以帶一些，因為有人會想換硬幣，記得準備給 guide 的禮物 ( 像是 Sole 人超好的 ! )
  • 信用卡 : 很好用，搭地鐵之類的也會需要，可以設定一下
  • 娃娃 : 我爸說放娃娃在電腦旁邊 EGOI 官方攝影師會比較想幫你拍照，你看楊岳蓁的海狸被拍那麼多張，我被拍到 $0$ 張

如果有人很傷心

當有人表現不好的時候，記得什麼都不要說 ( 對，如果你不小心說「沒關係啦，銀牌很棒耶」人家可能會不開心，你說「你打好爛，我很失望」人家會更傷心，不如什麼都別說 )，她可能需要一點獨處的時間，可能一兩天就會比較好了。

造成別人困擾

不要造成教授的困擾，心情不好的話可以講一下，不要自己不見 ( 我有認真檢討 )。
約定集合時間一定要守時，不然等你的人會生氣 ( 應該不是在酸法國隊遲到 $30$ 分鐘 )。

多拍照

不要想說別人會拍，人家可能拍了不傳給你 ( 李忠謀 )，想留下回憶的人事物記得把握機會多拍照。

心態

想去 EGOI 玩

我好像 $2025$ $9$ 月寫信給睿睿問說有沒有 EGOI，得到很含糊的回覆，反正我就當會有啦，開始抱著渺小的希望，甚至弄了個 blog 想記錄東西 ( 對這個 blog 的初衷就是想把這一系列 EGOI 東西記錄下來的喔 )。
直到 $12$ 月吧 EGOI 正式出現在 TOI 官網上，我就開始亂吵別人說有 EGOI 了喔，每天早上起床都在想我要當國手去國外玩哈哈哈，選上之後變成我想拿金牌。

我覺得準備 EGOI 的時候幾乎每天心情都特別好，把討厭的科展還是考試都丟一邊，在學校 vir 和檢討、去國培玩 BOTC 也很開心。
多享受過程滿重要的。

認真

在義大利的時候我有時會想到 : 等等我現在不是來玩的吧，是不是該認真一點，不要再玩/散步/跟別人聊天，要去做一點準備…
但比完之後我有時又會想到 : 等等我是不是把比賽看太重了，這樣對心態不太好吧，是不是要輕鬆一點，多跟別人玩/散步/聊天…

感覺在選訓我有時也會這樣想 ( 雖然我今年特別努力跟其他人玩，完全不後悔 )，只能說都出國了，打 code 就先不用吧，畢竟在國內都練習這麼多次，一定沒問題，就多多跟別人玩吧 ! 但還是要注意時間，留點時間整理心態給比賽。

留下美好的回憶

這是最重要的，做什麼事都要優先考慮這個喔。
雖然打不好但是我有時候想到 EGOI 2026 都還是會覺得很開心。

打不好

陳羿安跟我說數奧前國手之前在她去 EGMO 時有給她錦囊妙計，說打不好可以看，所以我先寫一個，大概是給今年的我看的，我要相信明年的我不會用到啦。

(｡-_-｡) 離 EGOI 結束也快一個月了，我可能也不是很擅長傷心很久 ( ? )，所以現在可以以一個比較平靜的心態來想這件事。 day2 pB 沒做出來無疑是這場比賽最大的遺憾，也接連影響之後的比賽節奏，但先撇除能力不足這件事不談，至少我在中後場有盡量調整節奏把 pC 該拿的部分拿完、pD 沒有 $0$ 分，到最後一刻也持續想拿到分數，這可能也是我在那時候能做的最多的事了。 總之要相信 day2 是一個意外，然後我打這麼爛下限也有銀牌欸。 所以再過一年一定會變很強很強對吧。

朋友

跟外國人多講話

都去國外了，當然要多跟外國人當朋友 ! ( 你看 Luca 和 Sole 這麼 nice )
我以前覺得我不是很會交朋友的人，不過我現在覺得其實還好，就只是看要或不要去做 ( ? )

回國後還有機會可以跟外國人繼續聊天，不過這可能要看雙方忙不忙和有沒有想講話，至於要怎麼 keep in touch，我也在研究，可能時不時分享生活趣事 ( 以及台灣美食 ) 滿不錯的。

講英文

大部分國家選手都會講英文，當然也可以跟日本人講日文，如果講得出來 ( 見下 )。
所以想跟人多交流就要練習講英文，我們國培 TOI 三模考完去吃河粉的時候有練習，但我跟楊岳蓁都在旁邊笑，只有陳瑀萱和鄧涵云有好好練。
另外有外國人在場的時候對自己人也講英文的話對方才不會有被 excluded 的感覺，這是 Luca 說的，我覺得很棒。

不過華人參賽者也很多就是了 ( 我大部分還是認識華人吧 )，希望下次多認識不會中文的外國人。

學外語

以下是一件很好笑的事 : 我八年級隨便學了一年的日文，程度大概是可以看懂日文的 JOI 題目，所以我決定去跟日本隊講話。
我 : こんにちは !
シンギ : こんにちは !
我 : よろしくお願いします !
シンギ : よろしくお願いします !
然後我們開始用英文講話，完蛋太好笑了。

我今年初原本想學一點義大利文，結果我的心態是我現在還沒當上國手，再說啦，當上之後變成我要多練習才能拿金牌，好忙再說啦，結果什麼都不會，感覺跟 guide 講一點她的語言會很有趣，真可惜。
所以我要來學波蘭文，明年就知道我是不是說說怪了。

後遺症

奧林匹亞回歸症候群

準備 EGOI 這段時間我幾乎就是在選訓跟學校圖書館兩邊跑，課業科展丟掉超開心，但在請了大概三個月的假回來之後真的在班上上課會有一個不舒服的感覺，每天都覺得很無聊，這時候就多多和朋友聊天吧 ( 如果好朋友江語萱跑去大學弄全國科展的話就只好來畫畫和找其他事做 )

學分保衛戰

為了準備 EGOI 我是高二下開學就沒碰過課業了，還用選訓 + EGOI 免掉兩次段考，所以三段一定要好好唸，不然會燒雞。
有些小科目像家政美術，同學可能很雷幫你在小組成績拿到 $9/35$ 加上有缺一些成績導致你總分只有 $57$ ，這時候一定要和老師好好溝通。
跟老師要打好關係，同學如果有什麼活動要稍微關心一下 ( 比如科學班成發要知道自己要幹嘛，簡報何時交 )，回國用餅乾賄賂是不錯的方法。

我沒有大學

我的 EGOI 和 APIO 兩面牌現在沒任何用，修法加油。

雜談

科學班

我當初就是為了不想考會考想說科學班資源比較多才來武陵科學班的，但說是這麼說呢，武陵苛學班在資訊競賽上的資源並沒有比較多，來武陵苛學班對你的資奧的優點大概只有 :

  • 班導人很好，公假很好請
  • 各科老師不知道資奧是啥 ( 數學老師在高一的時候會說「參加數物化生地科能競的同學加油喔」，是不是忘了什麼 )，但給予支持說會幫我調平常成績
  • 圖書館阿北看到我出現在討論室的時候以為科學班學生出現在這裡是理所當然，分不清楚我是公假還是翹課
  • 數學家同學 ( 雖然不是打資奧的 ) 跟我討論奇怪題目
  • 可愛的同學 ( 江語萱就是你 ) 陪我聊天

反正就是沒有競賽資源。
不過我現在覺得其實還好啦，可能是習慣了，我都自己練習 ( ? )，而且可以抽到好班導好老師好同學就很賺了，雖然武陵苛學班讓我感覺本質上仍是考試壓力鍋，然後還有專題 ( pensive )。

資讀

我都不去資讀，武陵的資讀根本沒什麼 ( 對高一的我來說 )。
以下是偏頗的個人觀點。

武陵資讀沒什麼資源的原因大概是 :

  • 沒什麼厲害的人，厲害的人走了也沒留下什麼東西
  • 進來的高一生有一點基礎的不多，所以資讀都要從基礎開始教
  • 少部分有基礎的高一生也沒有有系統 ( ? ) 的帶上去，沒有傳承
  • 不負責任的 EGOI 國手

( 但武陵資讀畢業的學姊在選訓當輔導員還有蕭凱鴻都對我很好。 )

然後現在我也沒有要去資讀做事 ( 我對它真的沒什麼感情 )，但如果有武陵的學弟妹需要幫忙，我應該會很樂意幫忙的。

不過我很喜歡 EGOI，所以我想來寫 EGOI 題解。

其他

沒意外我明年要再去一次 ( 金牌你給我等著 ! ! )
暑假應該會 vir OI 吧，正在想要不要找人一起打。
當然要看 IOI 幫我的偶像加油 !
希望我會寫 EGOI 題解，但我可能不是很會寫題解。
台灣是不是有人想出 EGOI 題目，太酷了 ! 但如果他出成功了是不是會跟我們一起去波蘭，感覺有點恐怖。
